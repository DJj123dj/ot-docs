## Quick Overview
The /close command is used to close a ticket. **It doesn't delete the ticket!**
You can give an optional reason to close it & you can always re-open the ticket!
**More info about ticket closing <InlineLink isdoc="true" url="docs/system/ticket-flow#closing">can be found here</InlineLink>!**

## Config
These are the <InlineLink url="docs/config/main" isdoc="true">configurable settings</InlineLink> for this command.
Some of them will make the command look different!
Other ones can disable the command or change how it works!

### <InlineLink url="docs/config/main" isdoc="true"><LabelText color="blue">config.json</LabelText></InlineLink> system/closeMode \{#config-closemode}
Using `system/closeMode`, you can change if **ðŸŸ¢ everyone** or **ðŸŸ  only admins** can close a ticket.
It's recommended to set it to `false` except if you have a lot of users using the close button without reason!

### <InlineLink url="docs/config/api" isdoc="true"><LabelText color="red">api.json</LabelText></InlineLink> disable \{#config-disable}
You can disable this command using the `api.json` file! This way, no-one can use this command!
This may be usefull if you want to replace a command by your own!
