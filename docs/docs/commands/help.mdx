## Config
These are the <InlineLink url="category/config" isdoc="true">configurable settings</InlineLink> for this command.
Some of them will make the command look different!
Other ones can disable the command or change how it works!

### <LabelText color="blue">config.json</LabelText> system/showSlashcmdsInHelp {#option-show-slashcmds}
Using `system/showSlashcmdsInHelp`, you can change the way commands are showed in the help menu!
You can choose for showing the **slash commands** or by showing the **text commands**

- Image 1: `true`
- Image 2: `false`
