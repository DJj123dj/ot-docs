## Quick Overview
The /help command is probably the first command you will try when working with open ticket.
It gives you a list with all the commands available in Open Ticket!
*But now that you are reading the docs, you can better check it here!*

## Config
These are the <InlineLink url="docs/config/main" isdoc="true">configurable settings</InlineLink> for this command.
Some of them will make the command look different!
Other ones can disable the command or change how it works!

### <InlineLink url="docs/config/main" isdoc="true"><LabelText color="blue">config.json</LabelText></InlineLink> system/showSlashcmdsInHelp \{#config-show-slashcmds}
Using `system/showSlashcmdsInHelp`, you can change the way commands are showed in the help menu!
You can choose for showing the **slash commands** or by showing the **text commands**

### <InlineLink url="docs/config/api" isdoc="true"><LabelText color="red">api.json</LabelText></InlineLink> disable \{#config-disable}
You can disable this command using the `api.json` file! This way, no-one can use this command!
This may be usefull if you want to replace a command by your own!
