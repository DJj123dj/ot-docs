`api.json` is a config file located in `./core/api/`! It's made for <ColorText color="blue">**developers who want to extend their bot with open ticket**</ColorText>!

With this config, you can disable certain features of open ticket. But you can also share the discord.js client & more!

:::danger warning
**This file is outdated & incomplete!** It's not recommended to edit or use any of the features.

For more info, join our **[discord server](https://discord.dj-dj.be)**
:::
