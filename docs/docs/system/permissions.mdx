On this page, you can find a list of the permissions required for commands, buttons & more actions in Open Ticket!
You will also get some information about where you can use it!

### Ticket Creation \{#creation}
Create a ticket using the embed or with `/ticket <id>`!

<table>
  <tr>
    <th>Permission</th>
    <td>🟢 Everyone</td>
  </tr>
  <tr>
    <th>Location</th>
    <td>🟠 Server, Ticket</td>
  </tr>
</table>

### Ticket Closing \{#closing}
