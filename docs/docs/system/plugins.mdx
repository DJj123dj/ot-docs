Open Ticket v3 has a small plugin system that you can use for lots of cool things! **Here are some examples:**
- Run custom code
- Add extra systems (moderation, giveaway, minigames,...)
- Respond to events (ticket creation, closing, deleting,...)

## Structure
Plugins are stored in `./plugins/` and are <ColorText color="red">**required to have the `.plugin.js` extension!**</ColorText>

If you want to know how to write plugins yourself, consider checking the developer wiki!
