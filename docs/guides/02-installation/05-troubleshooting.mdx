## ðŸ›  Troubleshooting \{#troubleshooting}
### `Unable to parse config ""..."` <M color="red">error</M> \{#config-parse-error}
This error is almost always produced when the JSON syntax is invalid.  
Try checking your config files with a code editor for possible issues.

<ScaleImage src="/img/troubleshooting/config-parse-error.png" alt="Failiure when loading general.json config." size="80%"/>

### `Cannot read property '...'  of undefined` <M color="red">error</M> \{#discordjs-missing}
Sometimes caused by invalid/uninstalled package versions. **Try re-installing all npm packages**.

<ScaleImage src="/img/troubleshooting/discordjs-missing.png" alt="Failed to load variables from discord.js because it wasn't installed." size="80%"/>

### `Syntax error: Unexpected token '?'` <M color="red">error</M> \{#nodejs-version-invalid}
This error is almost always produced when the `node.js` version is too low.  
Try re-installing node.js `v18` or above.

<ScaleImage src="/img/troubleshooting/nodejs-version-invalid.png" alt="Open Ticket console on config failure" size="80%"/>

### `Unexpected token ] in JSON at position ...` <M color="red">error</M> \{#unexpected-config-json}
This error is almost always produced when the JSON syntax is invalid.  
Try checking your config files with a code editor for possible issues.

<ScaleImage src="/img/troubleshooting/unexpected-config-json.png" alt="Open Ticket console on config failure" size="80%"/>
