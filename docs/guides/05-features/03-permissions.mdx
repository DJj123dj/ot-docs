Hello there! Welcome to the guide about <C color="blue">**Permissions**</C> in Open Ticket.
This guide explains how Open Ticket handles **command permissions**, how to customize them, and the default permissions for each command and ticket action.  

Permissions in Open Ticket determine **who can perform actions** like opening tickets, managing them, and using admin features.
You can customize these in the [`general.json` config file](/docs/reference/config/general.mdx#permissions).

## ⚙️ Configuring Permissions \{#configuration}
Permissions are set using the [**`"permissions"`** property](/docs/reference/config/general.mdx#system-permissions) in `general.json`.  
Each command or action has a **default permission**, but you can change it to fit your server’s needs.

### 🏷️ Available Permission Types \{#permission-types}
- **`"everyone"`** - Allow usage for everyone.
- **`"admin"`** - Allow only for `global admins` and `ticket admins`.
- **`"<role id>"`** - Allow for a specific Discord role ID.
- **`"none"`** - Disable the command/action entirely.  
<C color="red">**This will also remove it from the slash commands and help menu.**</C>

:::warning
Disabling commands by setting their permission to `"none"` will **not disable the buttons**.  
For disabling the buttons, there are other variables available in the config.
:::

### 👥 Who has access to `"admin"` permissions? \{#access-admin}
- ✅ Users with `Administrator` permissions.
- ✅ Users with a `globalAdmin` role.
- ✅ Users with a `ticketAdmin` role, but only in their assigned channels.

```json title="Default Permissions (general.json)"
//..."system"
"permissions":{
    "help":"everyone",
    "panel":"admin",
    "ticket":"none",
    "close":"everyone",
    "delete":"admin",
    "reopen":"everyone",
    "claim":"admin",
    "unclaim":"admin",
    "pin":"admin",
    "unpin":"admin",
    "move":"admin",
    "rename":"admin",
    "add":"admin",
    "remove":"admin",
    "blacklist":"admin",
    "stats":"everyone",
    "clear":"admin",
    "autoclose":"admin",
    "autodelete":"admin"
},
//..."system"
```
