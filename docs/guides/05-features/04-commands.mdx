## üéØ Command Overview \{#overview}
Below is a list of **all available commands** in Open Ticket, categorized by functionality.  

### üåê Global Commands \{#global}
The following commands can be used everywhere in the server. Some of them can even be used in DM.
| Command | Reference | Description |
|---------|-----------|-------------|
| **`/help`** | [**Full Explaination**](/docs/reference/commands/help.mdx) | View the help menu. |
| **`/panel`** | [**Full Explaination**](/docs/reference/commands/panel.mdx) | Spawn a ticket panel. |
| **`/ticket`** | [**Full Explaination**](/docs/reference/commands/ticket.mdx) | Directly create a ticket without using a panel. <C color="red">**(slash command only)**</C> |
| **`/clear`** | [**Full Explaination**](/docs/reference/commands/clear.mdx) | Delete multiple tickets at the same time using filters. |

### üé´ Ticket Management \{#management}
The following commands can only be used in valid ticket channels. They allow you to manage tickets easily.
| Command | Reference | Description |
|---------|-----------|------------|
| **`/close`** | [**Full Explaination**](/docs/reference/commands/close.mdx) | Close a ticket. |
| **`/reopen`** | [**Full Explaination**](/docs/reference/commands/reopen.mdx) | Reopen a closed ticket. |
| **`/delete`** | [**Full Explaination**](/docs/reference/commands/delete.mdx) | Delete a ticket permanently. |
| **`/claim`** | [**Full Explaination**](/docs/reference/commands/claim.mdx) | Claim a ticket as your own. |
| **`/unclaim`** | [**Full Explaination**](/docs/reference/commands/unclaim.mdx) | Unclaim a previously claimed ticket. |
| **`/pin`** | [**Full Explaination**](/docs/reference/commands/pin.mdx) | Pin the channel to the top of the category. |
| **`/unpin`** | [**Full Explaination**](/docs/reference/commands/unpin.mdx) | Unpin a previously pinned ticket. |
| **`/add`** | [**Full Explaination**](/docs/reference/commands/add.mdx) | Add a user to a ticket. |
| **`/remove`** | [**Full Explaination**](/docs/reference/commands/remove.mdx) | Remove a user from a ticket. |
| **`/rename`** | [**Full Explaination**](/docs/reference/commands/rename.mdx) | Rename the ticket channel. |
| **`/move`** | [**Full Explaination**](/docs/reference/commands/move.mdx) | Move the ticket to a different option type. |

### üìä Ticket Statistics \{#stats}
The following commands allow you to view statistics in the server, a ticket or from a user.
| Command | Reference | Description |
|---------|-----------|------------|
| **`/stats global`** | [**Full Explaination**](/docs/reference/commands/stats.mdx) | View global ticket statistics. |
| **`/stats ticket`** | [**Full Explaination**](/docs/reference/commands/stats.mdx) | View statistics of the current ticket. |
| **`/stats user`** | [**Full Explaination**](/docs/reference/commands/stats.mdx) | View ticket statistics for a specific user. |
| **`/stats reset`** | [**Full Explaination**](/docs/reference/commands/stats.mdx) | Reset ticket statistics. |

### üö´ Blacklist Management \{#blacklist}
The following commands allow you to view and manage the user blacklist.
| Command | Reference | Description |
|---------|-----------|------------|
| **`/blacklist view`** | [**Full Explaination**](/docs/reference/commands/blacklist.mdx) | View the current blacklist. |
| **`/blacklist get`** | [**Full Explaination**](/docs/reference/commands/blacklist.mdx) | Check if a user is blacklisted. |
| **`/blacklist add`** | [**Full Explaination**](/docs/reference/commands/blacklist.mdx) | Add a user to the blacklist. |
| **`/blacklist remove`** | [**Full Explaination**](/docs/reference/commands/blacklist.mdx) | Remove a user from the blacklist. |

### ‚åõ Auto Ticket Actions \{#auto-actions}
The following commands allow you to automatically manage, close & delete tickets.
| Command | Reference | Description |
|---------|-----------|------------|
| **`/autoclose enable`** | [**Full Explaination**](/docs/reference/commands/autoclose.mdx) | Enable auto-closing of inactive tickets. |
| **`/autoclose disable`** | [**Full Explaination**](/docs/reference/commands/autoclose.mdx) | Disable auto-closing of inactive tickets. |
| **`/autodelete enable`** | [**Full Explaination**](/docs/reference/commands/autodelete.mdx) | Enable auto-deletion of old tickets. |
| **`/autodelete disable`** | [**Full Explaination**](/docs/reference/commands/autodelete.mdx) | Disable auto-deletion of old tickets. |
