Open Ticket can be customised in all aspects of the bot! Here's a list of just a few of the things you can change:

### ğŸ“‹ Messages & Embeds \{#messages-embeds}
Open Ticket allows you to change **almost every message** the bot sends, including:  

- Ticket **panel messages** ([`panels.json`](/docs/reference/config/panels.mdx))
- Custom **welcome messages** when opening a ticket ([`options.json`](/docs/reference/config/options.mdx))
- Embed **colors, titles, descriptions, and footers** (All configs)
- The way **emojis** are used **in embed titles** ([`general.json`](/docs/reference/config/general.mdx))
- Error messages & other system messages ([`general.json`](/docs/reference/config/general.mdx))

:::tip advanced customisation

:::

### ğŸ›ï¸ Buttons, Dropdowns & Interactions \{#buttons-dropdowns-interactions}
Not every button is needed for every server. Open Ticket allows you to:  

- **Remove unnecessary buttons** like `Close` or `Claim` ([`general.json`](/docs/reference/config/general.mdx))
- **Change panel button colors** to match your theme ([`options.json`](/docs/reference/config/options.mdx))
- **Change system button colors** to match your theme ([Plugin Required](https://odplugins.dj-dj.be/tree/main/open-ticket/ot-customise-buttons))

:::tip advanced customisation
More advanced button, dropdown & interaction customisation is only possible with plugins.  
The default config only provides a basic amount of customisation.
:::

### ğŸ“¢ Commands \{#commands}
Don't need certain commands? **Disable them!**  

- You can **disable specific commands** that you don't want users to access. ([`general.json`](/docs/reference/config/general.mdx))
- This can be useful if you only want **certain features** of Open Ticket.

:::tip adding & modifying commands
To modify or add a new command, you will need to create a plugin.
A lot of new commands are already available in the [**official plugins repository**](https://odplugins.dj-dj.be/tree/main/open-ticket/)!
:::

### ğŸ“„ Transcripts\{#transcripts}
Ticket transcripts can be **adjusted** to fit your needs:  

- Choose between **HTML** and **plain text** transcripts  
- Modify the **layout & colors** of HTML transcripts  
- Control how and where transcripts are **saved**  

:::tip advanced customisation
Modifying Text Transcripts can be easily done via the **API using a plugin.**  
Modifying HTML Transcripts is currently **not possible**, but it will be possible in an upcoming version!
:::
