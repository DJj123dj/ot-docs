## Quick Overview
The /delete command is used to delete a ticket. **It permanently deletes the ticket!**
If you have configured transcripts in `transcriptconfig.json`, then they will also be created!
**More info about ticket deletion <InlineLink isdoc="true" url="docs/system/ticket-flow#deletion">can be found here</InlineLink>!**
