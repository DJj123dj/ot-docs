## Quick Overview
With the /ticket command, you can create a ticket without spawning an embed using the <InlineLink url="docs/commands/message" isdoc="true">/message command</InlineLink>!
The buttons & design can be configured in the <InlineLink url="docs/config/general" isdoc="true">config file</InlineLink>!
This is very usefull if you don't want to have a channel for tickets! And remember that this command can also be disabled!

## Config
These are the <InlineLink url="docs/config/general" isdoc="true">configurable settings</InlineLink> for this command.

Other ones can disable the command or change how it works!

### <InlineLink url="docs/config/api" isdoc="true"><M color="red">api.json</M></InlineLink> disable \{#config-disable}
You can disable this command using the `api.json` file! This way, no-one can use this command!
This may be usefull if you want to replace a command by your own!
