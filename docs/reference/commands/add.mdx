## Overview
Using the `/add` command, you can add someone to a ticket. This will give the user the same permissions as the creator of the ticket.
You can remove someone who was added manually using the `/remove` command.
You can also provide an optional reason. This will be shown in the logs.

- **Default Permissions:** ðŸŸ  Ticket Admins
- **Usable Locations:** ðŸ”´ Ticket Only
- **Text Command:** `!ticket add <user> [reason]`
- **Slash Command:** `/add <user> [reason]`

## Parameters
- **`<user>`:** A user in the server to add to the ticket.
- **`[reason]`:** A reason for adding a user to the ticket. **<C color="green">(Optional)</C>**
