## Overview
Using the `/blacklist` command, you can manage everything related to the user blacklist.
Add & remove users from the blacklist or take a look at all blacklisted users.
- **`view`:** View the current blacklist.
- **`get`:** Get details from a single user.
- **`add`:** Add a user to the blacklist. (With additional reason)
- **`remove`:** Remove a user from the blacklist. (With additional reason)

You can use these modes by inserting it as a subcommand.

- **Default Permissions:** ðŸŸ  Ticket Admins
- **Usable Locations:** ðŸŸ  Server (server & ticket)
- **Text Command:** `!ticket blacklist <view|get|add|remove> <user> [reason]`
- **Slash Command:** `/blacklist <view|get|add|remove> <user> [reason]`

## Parameters
- **`<view|get|add|remove>`:** Choose what to do with the blacklist system.
- **`<user>`:** The user to modify/view. **<C color="green">(Only when using `get`, `add` or `remove`)</C>**
- **`[reason]`:** A reason for changing the blacklist. **<C color="green">(Only when using `add` or `remove`)</C>**
