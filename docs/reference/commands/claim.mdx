## Overview
Using the `/claim` command, you can claim a ticket. This will let the support team know that you are handling this ticket.
If configured, the ticket can also be moved to another category based on the user who claimed it.
The user can also provide an optional reason. This will be shown in the logs.

:::warning upcoming feature
Currently admins are not removed when a ticket is claimed, but this will be possible in a future update.
:::
