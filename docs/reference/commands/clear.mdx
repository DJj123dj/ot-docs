## Overview
Using the `/clear` command, you can delete multiple tickets at the same time.
There is an optional filter available for filtering which tickets are affected.

- **Default Permissions:** ðŸ”´ Global Admins
- **Usable Locations:** ðŸŸ  Server (server & ticket)
- **Text Command:** `!ticket clear <id> [reason]`
- **Slash Command:** `/clear <id> [reason]`

## Parameters
- **`[filter]`:** A filter will affect which tickets are deleted. **<C color="green">(Optional)</C>**
    - `All` tickets
    - `Open` tickets only
    - `Closed` tickets only
    - `Claimed` tickets only
    - `Unclaimed` tickets only
    - `Pinned` tickets only
    - `Unpinned` tickets only
    - `Autoclosed` tickets only
