## Overview
Using the `/close` command, you can close a ticket. This will make the participants unable to write any new messages.
They are still able to reopen the ticket and use other buttons.
The user can also provide an optional reason. This will be shown in the logs.

- **Text Command:** `!ticket close [reason]`
- **Slash Command:** `/close [reason]`

## Parameters
- **`[reason]`:** A reason for closing the ticket. **<C color="green">(Optional)</C>**
