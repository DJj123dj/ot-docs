## Overview
Using the `/delete` command, you can delete a ticket. This will also generate a transcript when it is enabled in the config.
If configured, the ticket can also be deleted without transcript.
You can also provide an optional reason. This will be shown in the logs.

- **Default Permissions:** ðŸŸ  Ticket Admins
- **Usable Locations:** ðŸ”´ Ticket Only
- **Text Command:** `!ticket delete [reason]`
- **Slash Command:** `/delete [reason] [without-transcript]`

## Parameters
- **`[reason]`:** A reason for deleting the ticket. **<C color="green">(Optional)</C>**
- **`[without-transcript]`:** Delete without transcript. **<C color="green">(Optional)</C>**
