## Overview
Using the `/panel` command, you can spawn Open Ticket panels defined in the [`panels.json`](../config/panels.mdx) config.
There are many kinds of panels and they're highly customisable. More info can be found [here](../config/panels.mdx).

The panel has an `auto-update` feature that allows the panel to be updated automatically when the bot is restarted.

- **Default Permissions:** ðŸ”´ Global Admins
- **Usable Locations:** ðŸŸ  Server (server & ticket)
- **Text Command:** `!ticket panel <id> [auto-update]`
- **Slash Command:** `/panel <id> [auto-update]`

## Parameters
- **`<id>`:** A valid panel id.
- **`[auto-update]`:** Enable auto update on this panel. **<C color="green">(Optional)</C>**
