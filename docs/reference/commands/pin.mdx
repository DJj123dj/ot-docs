## Overview
Using the `/pin` command, you can pin a ticket. This will move the ticket to the top of the category and add a `ðŸ“Œ` emoji to the channel name.
This can be used to increase the priority of a ticket by making it more visible and easier to find.
You can also provide an optional reason. This will be shown in the logs.

- **Default Permissions:** ðŸŸ  Ticket Admins
- **Usable Locations:** ðŸ”´ Ticket Only
- **Text Command:** `!ticket pin [reason]`
- **Slash Command:** `/pin [reason]`

<ScaleImage src="/img/docs-v4/channel-pinned.png" alt="Example of a pinned channel." size="40%"></ScaleImage>

## Parameters
- **`[reason]`:** A reason for claiming the ticket. **<C color="green">(Optional)</C>**
