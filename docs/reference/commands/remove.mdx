## Overview
Using the `/remove` command, you can remove someone from a ticket. This will stop the user from being able to see and write in the ticket.
This won't work on admins or the ticket creator.
You can also provide an optional reason. This will be shown in the logs.

- **Default Permissions:** ðŸŸ  Ticket Admins
- **Usable Locations:** ðŸ”´ Ticket Only
- **Text Command:** `!ticket remove <user> [reason]`
- **Slash Command:** `/remove <user> [reason]`

## Parameters
- **`<user>`:** A user in the server to remove from the ticket.
- **`[reason]`:** A reason for removing a user from the ticket. **<C color="green">(Optional)</C>**
