## Overview
Using the `/rename` command, you can rename the channel name from a ticket.
You can also provide an optional reason. This will be shown in the logs.

- **Default Permissions:** ðŸŸ  Ticket Admins
- **Usable Locations:** ðŸ”´ Ticket Only
- **Text Command:** `!ticket rename [reason]`
- **Slash Command:** `/rename [reason]`

## Parameters
- **`[reason]`:** A reason for renaming the ticket. **<C color="green">(Optional)</C>**
