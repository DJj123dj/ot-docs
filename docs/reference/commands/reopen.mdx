## Overview
Using the `/reopen` command, you can reopen a ticket. This will make the participants able to write messages again.
This will only work when the ticket is already closed.
The user can also provide an optional reason. This will be shown in the logs.

- **Usable Locations:** ðŸ”´ Ticket Only
- **Text Command:** `!ticket reopen [reason]`
- **Slash Command:** `/reopen [reason]`

## Parameters
- **`[reason]`:** A reason for reopening the ticket. **<C color="green">(Optional)</C>**
