An event that emits when someone creates a ticket! It will emit with all modes: `button`, `dropdown` or `/ticket`!

## Parameters
|Parameter  |Type                                                                                            |Notes                                         |
|-----------|------------------------------------------------------------------------------------------------|----------------------------------------------|
|user       |[`discord.User`](https://discord.js.org/#/docs/discord.js/main/class/User)                      |The user that created the ticket              |
|channel    |[`discord.TextChannel`](https://discord.js.org/#/docs/discord.js/main/class/TextChannel)        |The ticket channel                            |
|guild      |[`discord.Guild`](https://discord.js.org/#/docs/discord.js/main/class/Guild)                    |The ticket server/guild                       |
|date       |[`Date`](https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Global_Objects/Date) |The date of ticket creation                   |
|ticketdata |<InlineLink isdoc="true" url="api/types/ticketdata">`openticket.TicketData`</InlineLink>        |Ticket Data (status, option, claim data, ...) |

## Example
```javascript title="example.plugin.js"
//log to console when someone creates a ticket!
events.onTicketOpen((user,channel,guild,date,ticketdata) => {
    console.log("hello, "+user.username+" opened a ticket!")
})
```
