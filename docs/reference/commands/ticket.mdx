## Overview
Using the `/ticket` command, you can create a ticket directly without having a panel.
The user can choose between the available ticket options. More info can be found at the [`options.json`](../config/options.mdx) config.

- **Text Command:** `!ticket ticket <id>`
- **Slash Command:** `/ticket <id>`

## Parameters
- **`<id>`:** A valid ticket option id.
