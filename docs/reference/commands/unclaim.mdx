## Overview
Using the `/unclaim` command, you can unclaim a ticket. This will let the support team know that you stopped handling this ticket.
The ticket will be moved to the original category when configured.
You can also provide an optional reason. This will be shown in the logs.

- **Usable Locations:** ðŸ”´ Ticket Only
- **Text Command:** `!ticket unclaim [reason]`
- **Slash Command:** `/unclaim [reason]`

## Parameters
- **`[reason]`:** A reason for unclaiming the ticket. **<C color="green">(Optional)</C>**
