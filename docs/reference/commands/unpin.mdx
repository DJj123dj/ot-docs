## Overview
Using the `/unpin` command, you can unpin a ticket.
The ticket will stay on the current position in the category, but the `ðŸ“Œ` emoji will be removed.
The user can also provide an optional reason. This will be shown in the logs.

## Parameters
- **`[reason]`:** A reason for unpining the ticket. **<C color="green">(Optional)</C>**
