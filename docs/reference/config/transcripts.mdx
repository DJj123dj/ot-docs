# transcripts.json <M color="cyan">object</M>
The `transcripts.json` file allows you to configure everything related to transcripts. There are currently 2 types of transcripts available (`text` & `HTML`)!
You can use these to save your ticket chats when they are deleted. More info about transcripts can be found [here](/docs/guides/system/transcripts.mdx).

<ScaleImage src="/img/docs-v4/transcript-html-v2.png" size="80%" alt="An example of the HTML Transcripts without custom background."></ScaleImage>

:::info HTML Transcripts FAQ
HTML Transcripts are currently hosted and processed at `https://transcripts.dj-dj.be`.

- <C color="red">**It is IMPOSSIBLE to change with the current version (`v2`)!**</C>
- <C color="green">**It will be POSSIBLE with the upcoming version (`v3`)!**</C>
- <C color="blue">**The processing of transcripts will ALWAYS stay on our servers because the compiler is closed-source.**</C>

Custom domains will probably be a paid feature, but it will be very cheap.
Suggestions are always welcome in our discord server!
:::

If you need help with configuring the transcripts, feel free to join our discord server!

## General Settings
These are the general settings for the transcripts. Here you can configure where transcripts are sent and where not.
<Divider></Divider>
### `"general"` <M color="cyan">object</M> \{#general}

Configure where the transcripts are sent. You are able to send it to a channel or the DM's of participants.

- **`"enabled"`:** <M color="red">boolean</M> - Enable the transcript system. <C color="red">**When this is disabled, nothing will work!**</C>
- **`"enableChannel"`:** <M color="red">boolean</M> - Send transcripts to a designated discord channel.
- **`"enableCreatorDM"`:** <M color="red">boolean</M> - Send transcripts to the creator of the ticket via DM.
- **`"enableParticipantDM"`:** <M color="red">boolean</M> - Send transcripts to all participants of the ticket via DM.
- **`"enableActiveAdminDM"`:** <M color="red">boolean</M> - Send transcripts to admins who actively sent messages in the ticket.
- **`"enableEveryAdminDM"`:** <M color="red">boolean</M> - Send transcripts to all admins assigned to the ticket, even if they didn't participate.
- **`"channel"`:** <M color="blue">string</M> - The discord channel ID where the transcript will be sent if `"enableChannel"` is enabled.
- **`"mode"`:** <M color="blue">string</M> - Set it to `"html"` for HTML Transcripts and `"text"` for text transcripts.

**✅ It's recommended to send transcripts to a channel & the creator of the ticket.**

<Divider></Divider>
## Embed Settings
These are additional settings for the embed which contains the transcript file or url.
<Divider></Divider>
### `"embedSettings"."customColor"` <M color="blue">string</M> \{#embed-custom-color}
**Default Value:** `"#f8ba00"` (Open Ticket Yellow)<br></br>
**Allowed Values:** `A valid hex-color`

The color of the embed which contains the transcript file or URL. Leave it empty to use the default color of the bot.

**✅ If you don't know how to get a hex-color, you can use the website [Color Picker](https://htmlcolorcodes.com/color-picker/) for it.**

<Divider></Divider>
### `"embedSettings"."listAllParticipants"` <M color="red">boolean</M> \{#embed-list-participants}
**Default Value:** `false` (Disabled)<br></br>

When enabled, everyone who sent at least 1 message will be listed in the transcript embed.

**✅ This list won't contain any role mentions from admins.**

<Divider></Divider>
### `"embedSettings"."includeTicketStats"` <M color="red">boolean</M> \{#embed-include-ticket-stats}
**Default Value:** `false` (Disabled)<br></br>

Enables including ticket statistics (e.g., duration, messages sent) in the transcript embed.

**✅ Not all statistics will be shown here. Only the most important ones.**

<Divider></Divider>
## Text Transcript Style
Configure the style of the text transcripts.
<Divider></Divider>
### `"textTranscriptStyle"."layout"` <M color="blue">string</M> \{#textstyle-layout}
**Default Value:** `"simple"`<br></br>
**Allowed Values:** `"simple"`, `"normal"`, `"detailed"`

Determines the level of detail in text transcripts.

- **`"simple"`:** Includes the text content and basic message details.
- **`"normal"`:** Includes everything from `"simple"` + embed titles & descriptions + attachments.
- **`"detailed"`:** Includes everything from `"normal"` + full embed details + reactions.

**✅ More details might make it more difficult to read.**

<Divider></Divider>
### `"textTranscriptStyle"."includeStats"` <M color="red">boolean</M> \{#textstyle-include-stats}
**Default Value:** `true` (Enabled)<br></br>

Enables the inclusion of ticket statistics in the text transcript.

**✅ This will include all available ticket statistics.**

<Divider></Divider>
### `"textTranscriptStyle"."includeIds"` <M color="red">boolean</M> \{#textstyle-include-ids}
**Default Value:** `false` (Disabled)<br></br>

Enables including user IDs in the text transcript.

**✅ Only enable this when needed. It might make transcripts harder to read.**

<Divider></Divider>
### `"textTranscriptStyle"."includeEmbeds"` <M color="red">boolean</M> \{#textstyle-include-embeds}
**Default Value:** `true` (Enabled)<br></br>

Enables including embeds in the text transcript.

**✅ Bots are the only users that can send embeds.**

<Divider></Divider>
### `"textTranscriptStyle"."includeFiles"` <M color="red">boolean</M> \{#textstyle-include-files}
**Default Value:** `true` (Enabled)<br></br>

Enables including file attachments in the text transcript.

**✅ This will also include a URL to the attachment. (when using `"normal"` or `"detailed"` layout)**

<Divider></Divider>
### `"textTranscriptStyle"."includeBotMessages"` <M color="red">boolean</M> \{#textstyle-include-bots}
**Default Value:** `true` (Enabled)<br></br>

Enables including messages from bots/apps in the text transcript.

**✅ Bots might send too much messages which could make the transcript unreadable.**

<Divider></Divider>
### `"textTranscriptStyle"."fileMode"` <M color="blue">string</M> \{#textstyle-file-mode}
**Default Value:** `"custom"`<br></br>
**Allowed Values:** `"custom"`, `"channel-name"`, `"channel-id"`, `"user-name"`, `"user-id"`

Determines the naming convention for the transcript file.

- **`"custom"`:** Use a custom filename set in `"customFileName"`.
- **`"channel-name"`:** Use the channel name as filename.
- **`"channel-id"`:** Use the channel id as filename.
- **`"user-name"`:** Use the creator username as filename.
- **`"user-id"`:** Use the creator id as filename.

**✅ All files will end with `.txt` as extension.**

<Divider></Divider>
### `"textTranscriptStyle"."customFileName"` <M color="blue">string</M> \{#textstyle-file-name}
**Default Value:** `""` (No custom name)<br></br>

A custom file name for the transcript file. Only used if [**`"fileMode"`**](#textstyle-file-mode) is set to `"custom"`.

**✅ The custom file name will have `.txt` as extension.**

<Divider></Divider>
## HTML Transcript Style
Configure the style of the HTML transcripts.
<Divider></Divider>
### `"htmlTranscriptStyle"."background"` <M color="cyan">object</M> \{#htmlstyle-background}

Configure a custom background for the HTML Transcripts. When disabled, the default background will be used.

- **`"enableCustomBackground"`:** <M color="red">boolean</M> - Enables a custom background for the HTML transcripts.
- **`"backgroundColor"`:** <M color="blue">string</M> - The background color. **(Default: <Hex hex="#36393e"></Hex>)**
- **`"backgroundImage"`:** <M color="blue">string</M> - A URL to a background image.

**✅ The background image will always overwrite the color.**

<Divider></Divider>
### `"htmlTranscriptStyle"."header"` <M color="cyan">object</M> \{#htmlstyle-header}

Configure a custom header for the HTML Transcripts. When disabled, the default header will be used.

- **`"enableCustomHeader"`:** <M color="red">boolean</M> - Enables a custom header for the HTML transcripts.
- **`"backgroundColor"`:** <M color="blue">string</M> - The background color of the header. **(Default: <Hex hex="#202225"></Hex>)**
- **`"decoColor"`:** <M color="blue">string</M> - The decoration (horizontal line) color of the header. **(Default: <Hex hex="#f8ba00"></Hex>)**
- **`"textColor"`:** <M color="blue">string</M> - The text color of the header. **(Default: <Hex hex="#ffffff"></Hex>)**

**✅ The header includes the name of the ticket & bot and is always visible.**

<Divider></Divider>
### `"htmlTranscriptStyle"."stats"` <M color="cyan">object</M> \{#htmlstyle-stats}

Configure custom stats for the HTML Transcripts. When disabled, the default stats will be used.

- **`"enableCustomStats"`:** <M color="red">boolean</M> - Enables custom stats for the HTML transcripts.
- **`"backgroundColor"`:** <M color="blue">string</M> - The background color of the stats. **(Default: <Hex hex="#202225"></Hex>)**
- **`"keyTextColor"`:** <M color="blue">string</M> - The text color of the stats labels. **(Default: <Hex hex="#737373"></Hex>)**
- **`"valueTextColor"`:** <M color="blue">string</M> - The text color of the stats values. **(Default: <Hex hex="#ffffff"></Hex>)**
- **`"hideBackgroundColor"`:** <M color="blue">string</M> - The background color of the stats hide button. **(Default: <Hex hex="#40444a"></Hex>)**
- **`"hideTextColor"`:** <M color="blue">string</M> - The text color of the stats hide button. **(Default: <Hex hex="#ffffff"></Hex>)**

**✅ The stats are visible on the right-side of the screen and can be hidden.**

<Divider></Divider>
### `"htmlTranscriptStyle"."favicon"` <M color="cyan">object</M> \{#htmlstyle-favicon}

Configure a custom favicon for the HTML Transcripts. When disabled, the default favicon will be used.

- **`"enableCustomFavicon"`:** <M color="red">boolean</M> - Enables custom stats for the HTML transcripts.
- **`"imageUrl"`:** <M color="blue">string</M> - The URL for the favicon image.

**✅ Using a small image **(32x32)** is recommended for better load time.**

# Example File
All variables above have been made acording to this config for Open Ticket `v4.0.0`. This configuration might change in future versions!
```json title="transcripts.json"
{
    "general":{
        "enabled":false,

        "enableChannel":false,
        "enableCreatorDM":false,
        "enableParticipantDM":false,
        "enableActiveAdminDM":false,
        "enableEveryAdminDM":false,

        "channel":"transcript channel id (or leave empty)",
        "mode":"html OR text"
    },
    "embedSettings":{
        "customColor":"#f8ab00 (or leave empty)",
        "listAllParticipants":false,
        "includeTicketStats":false
    },
    "textTranscriptStyle":{
        "layout":"simple OR normal OR detailed",
        "includeStats":true,
        "includeIds":false,
        "includeEmbeds":true,
        "includeFiles":true,
        "includeBotMessages":true,

        "fileMode":"custom OR channel-name OR channel-id OR user-name OR user-id",
        "customFileName":"this-is-a-transcript (or leave empty)"
    },
    "htmlTranscriptStyle":{
        "background":{
            "enableCustomBackground":false,
            "backgroundColor":"#f8ba00 (or leave empty)",
            "backgroundImage":"https://www.example.com/image.png (or leave empty)"
        },
        "header":{
            "enableCustomHeader":false,
            "backgroundColor":"#202225",
            "decoColor":"#f8ba00",
            "textColor":"#ffffff"
        },
        "stats":{
            "enableCustomStats":false,
            "backgroundColor":"#202225",
            "keyTextColor":"#737373",
            "valueTextColor":"#ffffff",
            "hideBackgroundColor":"#40444a",
            "hideTextColor":"#ffffff"
        },
        "favicon":{
            "enableCustomFavicon":false,
            "imageUrl":"https://transcripts.dj-dj.be/favicon.png"
        }
    }
}
```
