## Members
- **[`admin`](https://otgithub.dj-dj.be/tree/main/src/core/api/modules/permission.ts#L43):** Admin. Higher than a moderator. (Used for global-admins)
- **[`developer`](https://otgithub.dj-dj.be/tree/main/src/core/api/modules/permission.ts#L47):** Bot owner or all users from dev team. (Able to use all commands including `/stats reset`)
- **[`member`](https://otgithub.dj-dj.be/tree/main/src/core/api/modules/permission.ts#L37):** A normal member. (Default for everyone)
- **[`moderator`](https://otgithub.dj-dj.be/tree/main/src/core/api/modules/permission.ts#L41):** Moderator. Higher than the support team. (Unused)
- **[`owner`](https://otgithub.dj-dj.be/tree/main/src/core/api/modules/permission.ts#L45):** Server owner. (Able to use all commands including `/stats reset`)
- **[`support`](https://otgithub.dj-dj.be/tree/main/src/core/api/modules/permission.ts#L39):** Support team. Higher than a normal member. (Used for ticket-admins)
