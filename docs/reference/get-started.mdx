### üñ•Ô∏è Commands
Open Ticket provides a variety of commands to manage tickets. **Learn more:**
<FlexHorizontal>
<div style={{width:"50%"}} className="otdocs-flex-mobile-width">
- [**/help**](./commands/help.mdx) - Show the Open Ticket help menu.
- [**/panel**](./commands/panel.mdx) - Spawn a panel.
- [**/ticket**](./commands/ticket.mdx) - Create a ticket.
- [**/close**](./commands/close.mdx) - Close a ticket.
- [**/reopen**](./commands/reopen.mdx) - Reopen a ticket.
- [**/delete**](./commands/delete.mdx) - Delete a ticket. This will generate a transcript.
- [**/claim**](./commands/claim.mdx) - Claim a ticket.
- [**/unclaim**](./commands/unclaim.mdx) - Unclaim a ticket.
- [**/pin**](./commands/pin.mdx) - Pin a ticket.
- [**/unpin**](./commands/unpin.mdx) - Unpin a ticket.
</div>
<div style={{width:"50%"}} className="otdocs-flex-mobile-width">
- [**/add**](./commands/add.mdx) - Add a user to a ticket.
- [**/remove**](./commands/remove.mdx) - Remove a user from a ticket.
- [**/rename**](./commands/rename.mdx) - Rename the ticket channel name.
- [**/move**](./commands/move.mdx) - Move the ticket to a different option type.
- [**/clear**](./commands/clear.mdx) - Delete multiple tickets at the same time.
- [**/autoclose**](./commands/autoclose.mdx) - Manage the autoclose system.
- [**/autodelete**](./commands/autodelete.mdx) - Manage the autodelete system.
- [**/blacklist**](./commands/blacklist.mdx) - Manage blacklisted users.
- [**/stats**](./commands/stats.mdx) - View ticket system statistics.
</div>
