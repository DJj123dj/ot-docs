## How to use?
You can use a language by configuring the [**`"language"`** property](./config/general.mdx#language) in the `general.json` config.
You can choose between the **Config Values** in the list below.

### Custom Translation
If you want to add a language that doesn't exist yet or you want to translate the bot yourself,
you can edit the `./languages/custom.json` file and use `"custom"` as value in the config.
