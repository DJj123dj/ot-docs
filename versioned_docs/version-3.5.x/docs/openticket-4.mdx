Somewhere this year (2024), Open Ticket 4.0 will be released!
It's a completely rewritten bot with lots of new features.
It will also have the most advanced API & plugin system ever seen! You can edit, add & remove almost everything with plugins only!

## Features
- 📝 Completely rewritten from scratch!
- 💡 `class-based` programming! (instead of `function-based`)
- 🔌 200x more powerful plugins!
- 💬 Cool new embeds & messages!
- ⚫️ New `/blacklist` command!
- 📥 A completely revisited claim system!
- #️⃣ Numbered ticket names (`#ticket-1234`)
- 📁 Webhook-based log system
- And more...
